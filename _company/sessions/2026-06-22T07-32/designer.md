## 🖼️ 유튜브 썸네일 비주얼 에셋 제작 사양 (컨셉 2: The Void Exposure)

본 산출물은 'The Glitch Core' 미학, 'Deep Void/Neon Cyan' 팔레트를 극대화하여 클릭률(CTR)을 높이는 고해상도 유튜브 썸네일 비주얼 에셋 제작을 위한 **레이아웃 및 스타일 가이드**입니다.

### 1. 핵심 시각 사양 (Visual System Definition)

| 요소 | 설정 값 | 설명 |
| :--- | :--- | :--- |
| **핵심 미학** | The Glitch Core | 디지털 노이즈, 화면 깨짐(Glitch), 데이터 오류의 시각적 표현을 주 테마로 사용. |
| **메인 컬러 (배경)** | Deep Void (#05051A) | 극도로 깊고 어두운 네이비/블랙 계열. 공간감과 심연의 느낌 강조. |
| **액센트 컬러** | Neon Cyan (#00FFFF) | 전기적이고 차가운 시안색. 모든 텍스트와 핵심 요소에 적용하여 시각적 충격 유도. |
| **보조 컬러 (강조)** | Glitch Red/Magenta (#FF0066) | 오류 및 긴장감을 표현하는 붉은색 계열. 대비를 극대화하고 감정적 불안정을 추가. |
| **타이포그래피** | Monospace / Glitch Font | 제목(Headline)에는 기계적인 느낌의 모노스페이스(예: Space Mono, OCR A Extended 등)를 사용하되, 텍스트에 의도적인 글리치/왜곡 효과를 적용합니다. |

### 2. 레이아웃 및 구도 가이드 (Layout & Composition Guide)

**컨셉:** The Void Exposure (공허함 속에서 발견되는 극단적 대비와 불안정성)

*   **구도:** 중앙 집중형(Center-focused) 또는 대각선 분할 구도를 사용하여 시청자의 시선을 즉시 '균열' 지점으로 유도합니다.
*   **배경 처리:** 전체 배경은 **Deep Void (#05051A)**로 채우되, 이미지 내부에 미세한 디지털 노이즈(Grain/Noise) 필터를 적용하여 질감을 부여합니다.
*   **Glitch 효과 배치:** 썸네일의 **약 30% ~ 40% 영역**에 Neon Cyan과 Glitch Red/Magenta의 색상 분할을 사용하여, 화면이 깨지거나 데이터가 손상된 듯한 시각적 충격을 극대화합니다. 이 경계선은 날카롭고 불규칙해야 합니다.
*   **텍스트 배치:**
    *   **메인 헤드라인 (Title):** 중앙 상단 또는 중앙에 배치하며, Neon Cyan 색상으로 처리하고 굵게 강조합니다. 타이포그래피는 기계적인 느낌을 유지합니다.
    *   **서브 텍스트 (Hook/Question):** 배경의 어두운 영역(Deep Void)이나 Glitch 효과가 적용된 영역에 Glitch Red/Magenta로 배치하여 긴급함과 호기심을 자극합니다.

### 3. 디자인 실행 체크리스트 (Execution Checklist)

1.  [ ] **색상 적용:** 배경은 Deep Void, 강조는 Neon Cyan 및 Glitch Red를 명확히 분리하여 사용했는지 확인한다.
2.  [ ] **텍스처 추가:** Deep Void 배경에 미세한 디지털 노이즈/글리치 필터를 적용하여 'The Glitch Core' 미학을 구현했는지 확인한다.
3.  [ ] **대비 극대화:** Cyan과 Red의 대비가 시각적 긴장감을 최대화하고 있는지 검토한다. (불안 $\rightarrow$ 해방 공식에 부합하는 감정적 충격 유도)
4.  [ ] **폰트 스타일:** 사용된 텍스트가 기계적이면서도 불안정한 느낌을 주는지 확인한다.
